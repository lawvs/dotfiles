# Dotfiles

## Usages

```sh
sudo apt install -y git
git clone https://github.com/lawvs/dotfiles.git
cd dotfiles
sudo ./init.sh
./install.sh
```

Set and use Zsh as default shell

```sh
chsh -s $(which zsh)
```
